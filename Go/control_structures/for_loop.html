For Loop

The Go for loop is similar to—but not the same as—C's. It unifies for and while and there is no do-while. There are three forms, only one of which has semicolons.

  • The most basic type contains a single condition 
    
x := 5
for x < 100 {
  fmt.Println(x)
  x += 5
}
  • A classic initial/condition/after for loop. 
    
for i := 5; i < 100; i += 5 {
  fmt.Println(i)
}

Without any condition in which you use the break statement to exit the loop. 


stopAt := 4
current := 0

for {
  fmt.Println(current)
  current++

  //Exits the for loop after printing "4"
  if current == stopAt {
    break
  }
}

With a continue statement to skip to the next iteration 


// Print every number except 50
for i := 5; i < 100; i += 5 {
  if i == 50 {
    continue
  }
  fmt.Println(i)
}
