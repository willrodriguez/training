Arrays

An array is a numbered sequence of elements of a single type, called the element type. The number of elements is called the length of the array and is never negative. Arrays are useful when planning the detailed layout of memory and sometimes can help avoid allocation, but primarily they are a building block for slices.

NOTE: You should use slices instead.

  • Create an array 
    
var x [5]int
x[0] = 42
x[3] = 2
fmt.Println(x) //[42 0 0 2 0]
  • Create a multi-dimensional array 
    
var y [2][3]int
y[1][2] = 4
fmt.Println(y) //[[0 0 0] [0 0 4]]
  • Assign values at initialization 
    
primes := [6]int{2, 3, 5, 7, 11, 13}
fmt.Println(primes) //[2 3 5 7 11 13]
