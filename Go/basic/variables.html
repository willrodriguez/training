Variables

  • You can't have a variable and not use it or GO will raise an error.
  • A variable can only be of one type (statically typed language)
  • var declaration keyword is required if you're declaring a variable outside of a function body. Otherwise you can use := short declaration operator. 
    
var x = 5
var z int
var w int = 4

func main() {
  y := 10
  fmt.Println(w, x, y, z)
}
  • You can use _ character when receiving a return value that you won't use. 
    
num_bytes, _ := fmt.Println("Hello World") //num_bytes, error := fmt.Println("Hello World")
fmt.Println(num_bytes)
  • interface{} is the empty interface and every value is also of type interface{}
  • variable declaration 
    
x := 4 	//x = 4
var x int = 4 //x = 4
var x int //x = 0
var b, c int = 1, 2 //b = 1, c = 2
