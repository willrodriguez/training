Data Types

A boolean type represents the set of Boolean truth values denoted by the predeclared constants true and false.

Type Values Desc.
bool true/false Represents a truth value

A numeric type represents sets of integer or floating-point values.

A string type represents the set of string values. A string value is a (possibly empty) sequence of bytes. The number of bytes is called the length of the string and is never negative. Strings are immutable: once created, it is impossible to change the contents of a string.

Type Desc.
string Represents a set of string values. A string value is a (possibly empty) sequence of bytes.
