Constants

The const keyword lets the compiler know that the value of the identifier should not change from the initial value. A const can be declared at the package level or function level. 


const x = 5 // Will get its type when first used.
const z = 5 // A constant unlike variables won't generate an error if not used.
const y int = 4
const (
 a = 3
 b = 4
)

The iota keyword is used in const declarations to simplify definitions of incrementing numbers. Because it can be used in expressions, it provides a generality beyond that of simple enumerations.

  • The value is reset to 0 whenever it encounters a new keyword
  • It increments by 1 whenever it encounters a new line within a const block. 

type Weekday int

const (
 Sunday = iota + 1  //1
 Monday             //2
 Tuesday            //3
 Wednesday          //4
 Thursday           //5
 Friday             //6
 Saturday           //7
)
