Commands

  • go env: Shows you the go environment variables
  • go fmt: Formats a file to follow GO standard. If you do go fmt ./... it will go through every single file and format it.
  • go run: Executes the given go file "go run main.go"
  • go build: Builds an executable into the current directory go build main.go will build a main executable which you can then execute with ./main
  • go mod init [module]: Creates a new mod.go file, or see modules for more details.
