Web Server

An HTTP server is a program that listens for HTTP requests on a port of the machine with a static or dynamic IP address. The HTTP protocol is an internet protocol made up of TCP and IP protocols.

Go provides the net package that contains the utility packages to handle networking related stuff. This package contains the http package which is used to create HTTP servers and to make HTTP requests as a client. It also has the smtp package to send emails using SMTP protocol. 


package main

import (
  "fmt"
  "io"
  "log"
  "net"
)

func main() {
  li, err := net.Listen("tcp", ":8080")

  if err != nil {
    log.Panic(err)
  }

  defer li.Close()

  for {
    conn, err := li.Accept()

    if err != nil {
      log.Panic(err)
    }

    io.WriteString(conn, "\nHello from TCP Server\n")
    fmt.Fprintln(conn, "How is your day?")

    conn.Close()
  }
}
