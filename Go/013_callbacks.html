Callbacks

A callback is a function you pass as an argument to another function.

  • In the example below we iterate through a slice and use a callback function on every single item in the slice. 
    
func main() {
  s := []int{1, 3, 5}
  total := total(sum, s...)
  fmt.Println(total)
}

//take X and adds it to the total
func sum(x int, total int) int {
  return x + total
}

//takes a function "F" and calls it for every single item in the slice
func total(f func(y int, t int) int, x ...int) int {
  total := 0

  for _, v := range x {
    total = f(v, total)
  }

  return total
}
